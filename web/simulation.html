Симуляція: як фільми розкладаються по тижнях

Planeta Kino оновлює репертуар щочетверга. Тиждень прокату = чт → сер (7 днів). Нижче можна рухати «сьогодні» і бачити, куди потрапляє кожна прем'єра.

Правило в одному реченні

Для будь-якої дати d беремо її четвер-власник: відкочуємо d назад до останнього четверга (якщо сьогодні нд/пн/вт/ср — це четвер минулого кіно-тижня; якщо чт/пт/сб — це сам цей четвер). Усі фільми, чий premiere_date приземляється на той самий четвер-власник, належать до одного «тижня» у фільтрі «Скоро». 

// Thu = 4 в нотації JavaScript getDay()
function getThursdayOfWeek(date) {
  const d = new Date(date);
  d.setHours(0, 0, 0, 0);
  let diff = d.getDay() - 4;
  if (diff < 0) diff += 7;   // нд/пн/вт/ср → повертаємось на минулий чт
  d.setDate(d.getDate() - diff);
  return d;
}

Контролі

Четвер-власник: —

Оранжева лінія зверху = четвер (початок кіно-тижня). Блакитна крапка = є прем'єра цього дня.

4 наступні тижні та їхні фільми

Це демо-фільми — натисни +1 тиж чи обери іншу дату, щоб побачити, як вони переходять із Скоро у В прокаті.

Чому саме четвер?

  • Дистриб'ютори світового кіно синхронно випускають фільми у «cinema week», яка в Україні традиційно стартує у четвер.
  • Вкладка «В прокаті» у нашому дашборді показує фільми, які вже мають сеанс (section === "now") — незалежно від тижня.
  • Вкладка «Скоро» показує все з section === "soon" і дозволяє звузити до одного тижня через пігулки нагорі. Вони перераховуються динамічно від сьогоднішнього четверга-власника.
  • Якщо прем'єра минула, а фільм ще не отримав сеанс — у нашому статусі він стане running (компутабельний з дати), тож фільтр «Скоро» його вже не зачепить.
